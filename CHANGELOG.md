# Список изменений
Все значимые изменения в этом проекте будут отмечены в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект соблюдает [семантическое версионирование](https://semver.org/lang/ru/).

## [v0.10.0] - 2022-06-26
### Добавлено
- Механизм обработки ошибок.
- Генерация Swagger UI.
- Типы исключений ошибок слоя приложения.
### Изменено
- Методы контроллера теперь выполняются асинхронно.
- Добавлена базовый POM для проектов на Spring.
### Исправлено
- Вероятность ошибок в методах FakeUserRepository.

## [v0.9.0] - 2022-06-24
### Добавлено
- Краткое представление сущности пользователя (ответ на запрос) и тесты для неё.
- Представление сущности пользователя (ответ на запрос) и тесты для неё.
- Представление роли пользователя (ответ на запрос) и тесты для неё.
- Представление сущности пользователя (запрос) и тесты для неё.
### Изменено
- Переработаны ошибки доменного уровня.

## [v0.8.0] - 2022-06-18
### Добавлено
- Добавлена динамическая генерация тестовых данных.
### Изменено
- Доработана сущность пользователя.
- Доработана роль пользователя (добавлена валидация значений).
- Доработаны ошибки.
### Исправлено
- Методы для создания тестовых пользователей вынесены в отдельный класс и перенесены в модуль с тестами.

## [v0.7.0] - 2022-06-12

- Сценарии использования(создания, удаления, получения, обновления) уведомления.
