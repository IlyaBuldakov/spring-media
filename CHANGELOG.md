# Список изменений
Все значимые изменения в этом проекте будут отмечены в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект соблюдает [семантическое версионирование](https://semver.org/spec/v2.0.0.html).

## [v0.17.0] - 2022-07-26

### Добавлено
1. Функция вывода ленты контента.


## [v0.16.0] - 2022-07-24

### Добавлено
1. Получение параметров из файла контента.
2. Сохранение сущности контент в БД.
3. Функция удаления контента по идентификатору.


## [v0.15.0] - 2022-07-22

### Добавлено
1. Библиотека для работы с файлами Apache Tika.
2. Получение параметров из файла.
3. Поиск комментариев по идентификатору задачи.

### Изменено
1. Реализована загрузка файла в БД.


## [v0.14.0] - 2022-07-18

### Добавлено
1. Модель сущности контент для СУБД.
2. Имплементация репозитория сущности контент.
3. Представление сущности контент (ответ на запрос).
4. Представление сущности контент (запрос).
5. Представление сущности контенты (ответы на запрос).
6. Представление сущности тип контента (ответ на запрос).
7. Заготовка сценария создания сущности уведомления.
8. Заготовка сценария создания сущности задач. 
9. Модель сущности задач для СУБД.
10. Имплементация репозитория сущности задач. 
11. Модель сущности файлов для СУБД.
12. Имплементация репозитория сущности файлов.
13. Представление сущности файлов (ответ на запрос).
14. Представление сущности файлов (запрос).

### Изменено
1. Контроллер контента переработан в заготовку для работы с СУБД.
2. Контроллер задач переработан в заготовку для работы с СУБД.
3. Контроллер файлов переработан в заготовку для работы с СУБД.
4. Переработаны сценарии использования сущности контент.
5. Переработаны сценарии использования сущности файл.
6. Переработаны сценарии использования сущности задачи.


## [v0.13.0] - 2022-07-17

### Добавлено
1. Проверка уникальности email при создании пользователя.
2. Сценарий обновления сущности пользователя.

### Изменено
1. Для генерации Swagger UI теперь используется `springdoc-openapi` вместо `springfox`.
2. Переработана сущность контента в интерфейс.
3. Переработана сущность файла в интерфейс.
4. Переработана сущность задачи в интерфейс.
5. Переработаны методы в репозитории контента.
6. Переработаны методы в репозитории файла.
7. Переработаны методы в репозитории задачи.

### Исправлено
1. Восстановлена работа метода удаления пользователя по идентификатору в реализаци репозитория пользователей.


## [v0.12.0] - 2022-07-16

### Добавлено
1. Модель сущности комментарий для СУБД.
2. Имплементация репозитория сущности комментарий.
3. Представление сущности комментарий (ответ на запрос).
4. Представление сущности комментарий (запрос).

### Изменено
1. Представление сущности комментарий и ее репозиторий.
2. Контроллер для работы с сущностью комментарий переработан для работы с СУБД.
3. Сценарий создания сущности комментарий.


## [v0.11.0] - 2022-07-15

### Добавлено
1. Модель сущности уведомления для СУБД.
2. Имплементация репозитория сущности уведомления.
3. Метод создания сущности уведомления.
4. Представление сущности уведомления (ответ на запрос).

### Изменено
1. Представление сущности уведомления.
2. Контроллер для работы с сущностью уведомления переработан для работы с СУБД.


## [v0.10.0] - 2022-07-13

### Добавлено
1. Доработана обработка запросов в контроллерах.
2. Исключение для операций без авторизации.
3. Модель пользователя для СУБД.
4. Реализация репозитория пользователя в слое infrastructure.

### Изменено
1. Изменен интерфейс UseCase.
2. Версия JDK для поддержки preview-методов.


## [v0.9.0] - 2022-07-12

### Добавлено
1. Обработка ошибок слоя приложения.

### Изменено
1. Аттрибуты сущности пользователя и их валидация вынесены в отдельный тип данных.
2. Переработана сущность пользователя в интерфейс.
3. Тип данных Role, перенесен в интерфейс User.
4. Сценарии работы с сущостью пользователя.

### Удалено
1. Сценарий поиска пользователя.


## [v0.8.0] - 2022-07-11

### Добавлено
1. Генерация Swagger UI.
2. Представление сущности пользователя (ответ на запрос).
3. Представление сущности пользователя (запрос).
4. Представление роли пользователя (ответ на запрос).
5. Типы исключений ошибок слоя приложения.

### Изменено
1. Ошибки доменного уровня
2. Краткое представление сущности пользователя (ответ на запрос)

## [v0.7.0] - 2022-06-19

### Добавлено
1. Тесты сущности ошибки.

3. Краткое представление сущности пользователя (ответ на запрос) и тесты для неё.


### Изменено
1. Доработана проверка значений при создании сущности пользователя.
2. Доработаны тесты сущности пользователя.
3. Доработана геренерация пароля фейковых пользователей.

### Удалено

## [v0.6.0] - 2022-06-05

## [v0.5.0] - 2022-05-30

### Изменено
1. Добавлена динамическая генерация тестовых данных.
2. Доработана сущность пользователя.
3. Доработана роль пользователя (добавлена валидация значений).
4. Доработаны ошибки.

### Исправлено
1. Методы для создания тестовых пользователей вынесены в отдельный класс и перенесены в модуль с тестами.
2. Вспомогательные методы для работы с пользователями вынесены в тесты.


## [v0.4.0] - 2022-05-23

### Добавлено
1. Сценарий удаления пользователя по идентификатору и тесты для него.
2. Сценарий обновления данных пользователя и тесты для него.
3. Сценарий создания пользователя и тесты для него.
4. К проекту подключен Spring Boot.
5. В демонстрационных целях добавлены заготовки слоёв Application и Infrastructure.

### Изменено
1. К проекту добавлен Spring Boot.

### Исправлено
1. В классе `User` добавлены вспомогательные методы для создания тестовых пользователей.
2. Написана документация Javadoc.


## [v0.3.0] - 2022-05-22

### Добавлено
1. Сценарий получения списка всех пользователей и тесты для него.
2. Сценарий получения списка пользователей в соответствии с запросом.

### Изменено
1. Доработан сценарий получения пользователя по идентификатору и тесты для него.

### Исправлено
1. Код переоформлен в соответствии со стилем кодирования.


## [v0.2.0] - 2022-05-15

### Исправлено
